🎵 Spotify CD Overlay

Connect your Spotify to show a spinning CD with the album art of whatever you're currently playing. Use this as a Browser Source in OBS/Streamlabs.

  1. Go to Spotify Developer Dashboard and create an app
  2. Set the Redirect URI to your GitHub Pages URL, e.g.
    https://yourusername.github.io/spotify-cd-overlay/
  3. Copy your Client ID and paste it below
  4. Click Connect → log in to Spotify → you'll be redirected back here
  5. Once connected, copy the full page URL and use it as a Browser Source in OBS
